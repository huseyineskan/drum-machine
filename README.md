# Drum Machine

It is a simple drum application. Operation can be done via mouse or assigned keys. The sound being played appears on the screen instantly.

- (Q) Heater 1
- (W) Heater 2
- (E) Heater 3
- (A) Heater 4
- (S) Clap
- (D) Open-HH
- (Z) Kick-n'-Hat
- (X) Kick
- (C) Closed-HH
